🔧 配置 ImgBB API Key

按照以下步骤获取并配置你的 API Key

步骤 1：获取 API Key

1. 访问 ImgBB API 官网

2. 点击 "Get API key" 按钮

3. 注册或登录账号（可使用 Google 账号）

4. 填写应用名称并获取 API Key
