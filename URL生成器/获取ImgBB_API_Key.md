# 获取 ImgBB API Key 步骤

## 1. 注册 ImgBB 账号

访问：https://imgbb.com/

点击右上角 "Sign up" 注册账号（可以使用邮箱或 Google 账号）

## 2. 获取 API Key

1. 登录后，访问：https://api.imgbb.com/
2. 点击 "Get API key" 按钮
3. 填写应用名称（随便填，比如 "Image Uploader"）
4. 点击 "Get API key"
5. 复制生成的 API Key（格式类似：`1234567890abcdef1234567890abcdef`）

## 3. 配置到项目

获取 API Key 后，告诉我，我会帮你配置到代码中。

## ImgBB 特点

- ✅ 完全免费
- ✅ 图片永久有效（不会过期）
- ✅ 公网可访问
- ✅ CDN 加速
- ✅ 支持直链
- ⚠️ 单张图片最大 32MB
- ⚠️ 免费版有请求限制（但对个人使用足够）

## 生成的 URL 格式

上传成功后会得到类似这样的 URL：
```
https://i.ibb.co/xxxxxx/image.png
```

这个 URL 可以：
- 直接在浏览器打开
- 用于网页 <img> 标签
- 分享给任何人
- 永久有效
