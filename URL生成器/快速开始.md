# 🚀 快速开始指南

## 方式1：使用配置助手（推荐，最简单）

1. **双击打开 `config.html` 文件**
2. 按照页面提示获取 ImgBB API Key
3. 输入 API Key 并点击"生成配置代码"
4. 复制生成的代码
5. 打开 `script.js` 文件，找到第 65 行，粘贴替换
6. 保存文件
7. **双击打开 `index.html` 开始使用！**

## 方式2：手动配置

### 第一步：获取 API Key

1. 访问：https://api.imgbb.com/
2. 点击 "Get API key"
3. 注册/登录（可用 Google 账号）
4. 填写应用名称，获取 API Key
5. 复制 API Key（格式：`1234567890abcdef...`）

### 第二步：配置代码

1. 打开 `script.js` 文件
2. 找到第 65 行：
   ```javascript
   const IMGBB_API_KEY = 'YOUR_IMGBB_API_KEY_HERE';
   ```
3. 替换为：
   ```javascript
   const IMGBB_API_KEY = '你的API_Key';
   ```
4. 保存文件

### 第三步：开始使用

双击打开 `index.html` 文件，上传图片即可！

## 使用说明

### 上传图片

- **方式1**：点击"选择图片"按钮
- **方式2**：直接拖拽图片到上传区域

### 获取 URL

上传成功后：
1. 图片 URL 会自动显示在输入框中
2. 点击"复制"按钮一键复制
3. 这个 URL 是永久有效的公网链接

### URL 特点

✅ 永久有效（不会过期）
✅ 公网可访问（任何人都能打开）
✅ CDN 加速（全球快速访问）
✅ 支持直链（可用于网页）

示例 URL：
```
https://i.ibb.co/xxxxxx/image.png
```

## 常见问题

### Q: 上传失败怎么办？

A: 检查以下几点：
1. API Key 是否正确配置
2. 图片大小是否超过 32MB
3. 网络连接是否正常
4. 打开浏览器控制台（F12）查看详细错误

### Q: 可以删除已上传的图片吗？

A: 可以！上传成功后，在浏览器控制台（F12）会显示删除链接，保存好这个链接，访问即可删除图片。

### Q: 有上传数量限制吗？

A: ImgBB 免费版有请求频率限制，但对个人使用完全足够。

### Q: 可以上传什么格式的图片？

A: 支持常见图片格式：JPG、PNG、GIF、BMP、WebP 等。

### Q: 需要运行服务器吗？

A: 不需要！直接双击 `index.html` 即可使用。`server.js` 是可选的本地服务器。
