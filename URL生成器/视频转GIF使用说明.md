# 🎬 视频转GIF表情包制作工具

## 功能介绍

这是一个完全在浏览器中运行的视频转GIF工具，无需上传到服务器，保护隐私！

### ✨ 主要功能

1. **视频上传** - 支持拖拽和点击上传
2. **时间裁剪** - 精确选择开始和结束时间
3. **尺寸调整** - 自定义GIF宽度
4. **帧率控制** - 调整流畅度和文件大小
5. **实时预览** - 转换前预览视频
6. **一键下载** - 下载生成的GIF
7. **上传图床** - 自动上传到ImgBB获取永久链接

## 🚀 使用方法

### 1. 打开工具
双击打开 `video-to-gif.html` 文件

### 2. 上传视频
- 点击"选择视频"按钮，或
- 直接拖拽视频文件到上传区域

### 3. 调整参数

**时间设置：**
- 开始时间：GIF从视频的哪个时间点开始
- 结束时间：GIF在视频的哪个时间点结束
- 建议时长：2-5秒（文件更小，加载更快）

**尺寸设置：**
- 宽度：GIF的宽度（像素）
- 建议：300-500像素（适合聊天使用）
- 高度会自动按比例调整

**帧率设置：**
- FPS：每秒帧数
- 建议：10-15 FPS（流畅且文件小）
- 越高越流畅，但文件越大

### 4. 开始转换
点击"🎨 开始转换"按钮，等待处理完成

### 5. 下载或分享
- 点击"💾 下载GIF"保存到本地
- 点击"☁️ 上传到图床"获取永久链接

## 🔧 技术实现

### 核心技术
- **FFmpeg.wasm** - 在浏览器中运行的FFmpeg
- **完全客户端处理** - 视频不上传到服务器
- **高质量转换** - 使用palette优化色彩

### 转换流程
1. 加载FFmpeg.wasm库
2. 读取视频文件到内存
3. 使用FFmpeg命令处理视频
4. 生成优化的GIF文件
5. 提供下载或上传选项

### 优化算法
```
-vf fps=15,scale=480:-1:flags=lanczos,split[s0][s1];[s0]palettegen[p];[s1][p]paletteuse
```
- `fps=15` - 设置帧率
- `scale=480:-1` - 设置宽度，高度自动
- `lanczos` - 高质量缩放算法
- `palettegen` - 生成调色板
- `paletteuse` - 使用调色板优化色彩

## ⚠️ 注意事项

### 文件大小限制
- 建议视频文件小于100MB
- 生成的GIF建议小于5MB
- 太大的GIF加载慢，不适合分享

### 浏览器要求
- 推荐使用Chrome、Edge、Firefox最新版
- 需要支持WebAssembly
- 需要足够的内存（建议4GB以上）

### 处理时间
- 短视频（<10秒）：约10-30秒
- 中等视频（10-30秒）：约30-60秒
- 长视频（>30秒）：不建议，建议先裁剪

### 隐私保护
- ✅ 所有处理在浏览器本地完成
- ✅ 视频不会上传到任何服务器
- ✅ 只有生成的GIF会上传到图床（可选）

## 🐛 常见问题

**Q: 转换很慢怎么办？**
A: 
- 减少时长（建议2-5秒）
- 降低宽度（建议300-500像素）
- 降低帧率（建议10-15 FPS）

**Q: 生成的GIF太大怎么办？**
A:
- 缩短时长
- 减小尺寸
- 降低帧率

**Q: 画质不好怎么办？**
A:
- 增加宽度
- 提高帧率
- 使用高质量的源视频

**Q: 浏览器卡死了？**
A:
- 视频文件可能太大，建议先压缩
- 关闭其他标签页释放内存
- 刷新页面重试
