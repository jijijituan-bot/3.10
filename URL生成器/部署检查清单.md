## 🔧 Railway 部署步骤

### 方法1: GitHub 部署（推荐）
1. **推送代码到GitHub**
   ```bash
   git add .
   git commit -m "Railway部署配置完成"
   git push origin main
   ```

2. **Railway控制台操作**
   - 访问 https://railway.app
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择你的仓库
   - 等待自动部署完成

### 方法2: Railway CLI 部署
```bash
# 安装CLI
npm install -g @railway/cli

# 登录
railway login

# 初始化
railway init

# 部署
railway up
```

## 🚨 常见问题排查

### 问题1: 部署失败
**检查项目**:
- 确认 `package.json` 中的 `start` 脚本
- 检查 Node.js 版本兼容性
- 查看 Railway 部署日志

### 问题2: 应用无法访问
**检查项目**:
- 确认端口配置 `process.env.PORT`
- 检查主机绑定 `0.0.0.0`
- 访问 `/health` 端点测试

### 问题3: 静态文件404
**检查项目**:
- 确认文件路径正确
- 检查 MIME 类型配置
- 验证文件权限

### 问题4: API功能异常
**检查项目**:
- 确认 ImgBB API Key 正确
- 检查 CORS 配置
- 测试网络连接
